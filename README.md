# Jijau-Medical-And-General-Store

Modern, responsive Single Page Application (SPA) website for Jijau Medical And General Store, built with React and Tailwind CSS.

## Setup Instructions

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```

## Features
- Mobile-first responsive design
- Clean Healthcare aesthetic
- Embedded WhatsApp & Call-to-action integrations
- SEO & Meta tags optimized
- Lightbox gallery for store images
